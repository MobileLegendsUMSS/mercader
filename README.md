# Mercader App

## Requisitos previos
- JDK 17
- IntelliJ IDEA
- Instalar Android SDK API 36 - Android 16
- Instalar los Plugins Android Design Tools y Android
- Imagen de Google Pixel 6 (Recomendado)

## Configuración
1. Clonar el repositorio
2. Abrir el proyecto en IntelliJ IDEA
3. Sincronizar Gradle
4. Ejecuta en emulador o dispositivo fisico

## Estructura del proyecto
La estructura de carpetas está generada automáticamente al crear
un proyecto con Android desde IntelliJ Idea

## Sincronizar Gradle
Usar el botón del elefante (Sync Project with Gradle Files)

## Configurar JDK
- Instalar JDK 17.0.2 desde https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html
- File > Project Structure > Project
- Seleccionar JDK 17

## Configurar emulador
- Tools > Android > AVD Manager
- Create Virtual Device > Pixel 6 > Next
- Seleccionar API 36 (Android 16) > Download
- Finish
